# Priyanshu Gupta - MBA & PGDM Project Assistance Website

A premium, high-converting, and professionally polished static portfolio website designed for **Priyanshu Gupta**—an Online PGDM Graduate from Welingkar Institute (WeSchool) who provides academic project report formatting, presentation slide design, and documentation mentorship to MBA/PGDM students.

Designed as a premium, high-converting lead generation platform, the site primarily channels student traffic into customized, pre-formatted WhatsApp and email academic quotes.

---

## 🌟 Key Features

- **Executive Aesthetic:** Beautiful business-school styling built using custom Navy, Blue, and Gold color harmony with fluid, responsive layouts.
- **Frosted Glass Navigation:** Interactive sticky header with blur-backdrop filters that shifts themes seamlessly as students scroll through pages.
- **Why Choose Us Grid:** Clean, interactive service-benefit panels constructed with customized inline SVG icons.
- **Sample Work Showcase:** Responsive, category-filterable gallery supporting reports, slides, and layouts with an integrated scroll-locked Lightbox modal viewer.
- **Interactive FAQ Accordions:** Sleek, lightweight sliding answer drawers engineered entirely in Vanilla JavaScript without heavy external frameworks.
- **Smart WhatsApp Redirection:** Advanced form verification that automatically compiles students' names, specialization programs, topics, and numbers into a beautifully readable text format ready for instant WhatsApp messaging.
- **Responsive Layout:** Formats cleanly across standard monitors, tablets, and mobile phone breakpoints down to 480px.

---

## 🛠️ Technology Stack

- **Markup:** HTML5 (clean semantic outline, full SEO tags, and open-graph configurations)
- **Styling:** Vanilla CSS3 (dynamic HSL tokens, customized typography mappings, micro-interaction elevations)
- **Interactions:** Vanilla JavaScript (lightweight event listeners, active states, custom validators)
- **Visuals:** SVG vector icons (crisp resolution, fast loading speeds)

## 🚀 Running Locally

Since this is a fully static website, it does not require complex build engines or compilers. You can run it locally in two simple ways:

### Method 1: Direct File Opening
1. Download or clone this repository.
2. Double-click the `index.html` file to open it instantly in any modern web browser.

### Method 2: Lightweight Dev Server
To test form submissions and live layouts exactly as they would behave in production, start a simple HTTP server:
```bash
# Using Node.js npx
npx http-server -p 3002

# Or using Python 3
python3 -m http.server 3002
```
Once started, visit **`http://localhost:3002`** in your browser.
